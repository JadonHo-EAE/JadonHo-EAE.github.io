# EAE 2026 Portfolio — Jadon Ho Cheok Chun

**Live site:** https://JadonHo-EAE.github.io

My application portfolio for the Diploma in Cybersecurity & Digital Forensics
(Poly EAE 2026). Built as a single self-contained HTML file — all fonts,
certificates and evidence are embedded, so it runs offline and has zero
external dependencies.

## What's inside

- Four verified certifications (TryHackMe Pre-Security & Cyber Security 101,
  Cisco Introduction to Cybersecurity & Introduction to IoT) with an in-page
  viewer — click any card
- A case file on my local Llama 3 (8B) deployment with Ollama, including the
  Task Manager evidence of the RAM bottleneck I diagnosed (38% → 90%)
- A live interactive terminal — type `help` in the Contact section
- My journey: class Cyber Wellness Ambassador, National School Games badminton
  champion (C Div League 3, 2024), and the road to this application

## How it was built

I directed this project end to end: concept, structure, all content and
evidence, sourcing UI/UX reference repos from GitHub, and QA on every build.
The visual system was designed in Claude Design and the code generated with
Claude Code, working from my prompts across many revision rounds. Full
transparency is on the page itself — see the **Build Log** section (or type
`credits` in the terminal).

Every certificate, date, screenshot and stat on the page is real and
verifiable.
